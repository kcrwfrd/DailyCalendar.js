# Daily Calendar
A simple day-view event calendar.

## Dependencies
* Underscore or LoDash, for rendering templates
* jQuery

## Development
Clone the repo, then `npm install && bower install && grunt develop` (also depends on the Compass gem being installed)

## Changelog
